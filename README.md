## 课程简介

AiforU 是一门面向所有人的 AI 基础培训课程。从认知到实践，从原理到构建，6 大模块 27 节课，帮助你建立 AI 时代的工作新思维，成为可 AI 驱动的超级个体。

## 课程结构

| 模块 | 主题 | 课程数 |
|------|------|--------|
| 魂篇 | 重新认识 AI，理解发展趋势与能力边界 | L1-L3 |
| 骨篇 | AI 核心原理：Token、向量、LLM、多模态、RAG | L1-L6 |
| 身-应用篇 | 与 AI 高效协作的实战方法论与技巧 | L1-L6 |
| 皮-构建篇 | 业务流程 AI 化，经验沉淀与自动化 | L1-L3 |
| SKILL 篇 | AI 能力的最小封装单元 | L1-L3 |
| AGENT 篇 | 从对话到行动，AI Agent 的完整世界 | L1-L6 |

**6 大模块 · 27 节课 · 150+ 知识点 · 27 张图解 · 可复制 Prompt 模板**

## 课程特色

- **白牌通用**：适用于任何企业或个人的 AI 培训
- **图文并茂**：浅色主题，SVG 图解，阅读体验友好
- **实战导向**：每节课配 Prompt 模板，即学即用
- **前沿信息**：包含 2026 主流 AI 大模型/工具对比（Claude、GPT、Gemini、DeepSeek 等）
- **零依赖**：单 HTML 文件，浏览器直接打开，无需安装

## 部署到 GitHub Pages

1. 将仓库推送到 GitHub
2. 进入 **Settings → Pages**
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 点击 **Save**，等待几分钟后即可通过 `https://你的用户名.github.io/仓库名/AiforU.html` 访问
